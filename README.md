# Agentic Prisoner’s Dilemma

An agentic social experiment grounded in the Prisoner’s Dilemma, designed to study how autonomous AI agents with different personalities and strategies converge or fail to converge toward cooperation.

## Overview

This project simulates populations of AI agents interacting through repeated dilemma scenarios.

Each agent is defined by:

- a personality profile such as altruistic, selfish, deceptive, or risk-averse
- a decision strategy such as tit-for-tat, always cooperate, always defect, or adaptive
- memory and reasoning capabilities

Agents can communicate before making decisions, enabling negotiation, signaling, and deception.

The goal is to observe how cooperation emerges, collapses, or stabilizes under different conditions.

## Why this matters

The Prisoner’s Dilemma captures a fundamental tension between individual incentives and collective outcomes.

This experiment extends classic game theory into a new setting:
LLM-powered agents with language, identity, and adaptive reasoning.

Understanding these dynamics is critical for:

- multi-agent AI systems
- decentralized governance
- mechanism design and token economies

## Key Features

- Heterogeneous AI agents with distinct personalities
- Strategy diversity including fixed and adaptive behaviors
- Pre-play communication between agents
- Iterated interactions with memory
- Optional reputation systems and persistent identity
- Experiment configuration for different incentive structures

## Research Questions

- Under what conditions does cooperation emerge
- How does communication impact trust and outcomes
- Which strategies dominate over time
- How resilient is cooperation against manipulation
- What role does reputation play in stabilizing behavior
